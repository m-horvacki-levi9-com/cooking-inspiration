# Cooking Inspiration

Cooking Inspiration is a project for discovering weekend meal ideas, with Bring! shopping-list integration and localization support.

## Project Context

This repository is an experiment for testing GitHub Copilot app workflows as part of a reasoning system that includes both humans and AI agents.

Contributors are expected to collaborate with both people and custom agents during planning, refinement, architecture, and implementation.

## Local Environment Requirements

Before contributing, make sure your machine has the following:

- Git (latest stable)
- PowerShell 7+
- Node.js 20+
- npm (comes with Node.js)
- Visual Studio Code (latest stable)
- Visual Studio (latest stable, for backend development)

## Recommended Developer Setup

- GitHub Copilot extension enabled in Visual Studio Code
- Access to this repository and its GitHub Wiki
- Ability to run terminal commands from project root

## Who Should Join

This project is a good fit for developers who want to:

- Build software in a human-plus-agent collaboration model
- Practice structured refinement and architecture workflows
- Work with clear roles across delivery, architecture, frontend, and backend
